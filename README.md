# QuizCheater 🚧

> **Status: halbfertig / nicht vollendet.** Dieses Projekt wurde nie fertiggestellt und ist
> im aktuellen Zustand nicht produktiv nutzbar — es bleibt als Projekt-Einblick online.

## Worum geht's?

Ein Quiz-Tool, angelehnt an das **„WER LÜGT HIER?“**-Format von
[Vlesk](https://www.youtube.com/@Vlesk), gebaut für den Streamer
[umut_rre](https://www.twitch.tv/umut_rre).

Idee: Kandidaten beantworten Quizfragen — einer davon blufft. Das Tool sollte das
Format-Setup (Fragen, Runden, Auswertung) für Streams unterstützen.

## Tech

- Angular 16 (Angular CLI 16.1.4)
- `ng serve` → http://localhost:4200
